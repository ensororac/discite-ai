# Discite AI

> *Discite* — Latin for "learn"

An interactive platform for K–10 students to explore how Large Language Models really work — from tokenisation to prediction.

## Six Modules

| Module | Title | Year Bands |
|--------|-------|-----------|
| M1 | Tokenisation | 3–10 |
| M2 | Embeddings | 5–10 |
| M3 | Attention | 7–10 |
| M4 | Transformer Architecture | 9–10 |
| M5 | Prediction | 5–10 |
| M6 | Data & Bias | 3–10 |

## Tech Stack

- **Framework:** Next.js 15 (App Router, TypeScript)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Hosting:** Cloudflare Pages (static export)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build & Deploy

```bash
npm run build
```

Deploys automatically to Cloudflare Pages on push to `main`.
